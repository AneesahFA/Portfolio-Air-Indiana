Artifact: Air Indiana Chatbot

Title: Air Indiana Chatbot Project

Objective: To learn how to design, develop, and deploy a conversational AI chatbot tailored to airline customer service needs.

Description: This chatbot simulates a virtual assistant for Air Indiana, guiding users through flight inquiries, booking steps, and general support. The chatbot responds intelligently to user input using ChatGPT’s built-in conversational capabilities.

Live Demo: Try the Air Indiana Chatbot

Process: I used ChatGPT to create and train the chatbot by feeding it sample questions and evaluating its responses. I refined its training data by asking ChatGPT how it would respond to various user inputs, effectively teaching it how to handle common airline-related inquiries. The chatbot uses its own AI capabilities along with the contextual guidance I provided to deliver accurate and engaging responses.

Tools & Technologies: ChatGPT, prompt design, testing iterations, GitHub Pages for portfolio deployment

Value Proposition: Demonstrates my ability to build AI-driven user experiences, reflecting both technical and soft skills like communication and iteration.

Unique Value: This project shows how AI and user-driven data training can be combined to enhance digital customer service in the airline industry.

Relevance: Aligned with learning objectives in software development, AI, and digital transformation.
